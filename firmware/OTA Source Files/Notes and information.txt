Notes and information

A few things to note:

I had to get the final URL of the github files. 
First I locate the files in github and grab their URL. Then, I find the final URL of that original URL.
To do so I used the following command in a terminal:
"curl -v https://github.com/Derek-Gutheil/ribbit-network-frog-sensor/raw/feature/OTA/firmware/OTA/test_bin.ino.bin"

Doing so for the above example returns the following: "https://raw.githubusercontent.com/Derek-Gutheil/ribbit-network-frog-sensor/feature/OTA/firmware/OTA/test_bin.ino.bin"

Then, I follow the instructions laid out here: https://esp32.com/viewtopic.php?f=2&t=15788&hilit=0x2700#p61358
Specifically, I direct firefox to the url that curl returned, then I press the padlock, connection secure, more information, view certificates, I used the *.github.io one, and pressed "PEM (chain)" to download a file called "github-io-chain.pem". Then I was able to extract the middle of 3 certificates which looked like so:
